# PickleCrew Project Brief

## What is PickleCrew?
PickleCrew is a full-stack web application (picklecrew.net) for recreational pickleball communities. It replaces messy group chats, spreadsheets, and manual scorekeeping with a polished, mobile-first platform.

**Tagline:** "Your Pickleball Crew, Leveled Up"

## Core Goals
1. **Game Tracking** — Log games, track scores, maintain win/loss records
2. **Community Building** — Organize events, build groups/crews, find players
3. **Performance Analytics** — ELO ratings, win rates, streaks, leaderboards
4. **Friendly Rivalries** — Challenges, rivalry tracking, hot streaks (🔥)
5. **Celebration** — Badges, achievements, Golden Pickle (11-0 shutout)

## Products / Surfaces
1. **Mobile App** — Primary surface; full-featured iOS/Android-style experience via responsive web
2. **Desktop Web** — Same app, wider layout; primarily used for analytics, admin dashboard
3. **Open Play Board** — Public-facing page, no login required
4. **Admin Dashboard** — Role-gated internal tool
